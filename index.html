LOGO! 0BA6 Program Dumper

Reads the circuit-program memory off a Siemens LOGO! 0BA6 over the serial PG protocol, in your browser. Desktop uses Web Serial; Android Chrome uses WebUSB. Nothing is uploaded — it all runs locally.

Before you start — put the LOGO! in STOP (ESC → Stop on the device). Plug the USB cable in. On Android use an OTG adapter and Chrome. This is a first-cut, hardware-untested tool: watch the log, and if the program area comes back wrong, the "program name" self-test below tells you whether the address page is right.
not connected

Defaults assume the 0BA5 program map (0x0EE8–0x16B7, 2000 bytes) under the 0BA6 0x00FF____ page. The name self-test reads 0x00FF0570 (16 bytes) — if it prints readable text, the page mapping is correct and the dump address is trustworthy. If not, the mapping needs adjusting; tell me what the self-test returned.

Reads the pointer table (0x0C14), output/merker wiring (0x0E20) and program memory (0x0EE8), then walks every block and prints a human-readable netlist. Basic gates (AND/OR/NOT/…) are decoded exactly; special-function wiring is resolved but their numeric parameters (timer values etc.) are shown raw for now.
